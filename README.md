### This project is a modular monolith created to study this subject.

- Programming language: Typescript;
- Database: SQLite;
- ORM: Sequelite

Made unit tests, integration tests, and end-to-end tests.

### To run
- Clone the project: `git clone https://github.com/mateuszimmer/monolito-modular.git`
- Install dependencies: `npm install`
- You can run tests with: `npm run test`
- You can make your tests using: `npm run dev`
-- it will run in dev mode and create a new folder called testes in the project root folder and a SQLite file for the dev database.
